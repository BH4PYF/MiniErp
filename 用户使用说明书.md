## 1. 系统概述

### 1.1 系统简介

MiniErp 项目管理系统是一款专业的工程项目全流程管理系统，集成了材料管理、分包管理、合同管理、进度计量、结算管理等核心功能，为工程项目提供一站式解决方案。

**系统定位：**
- 面向中小型建筑工程项目
- 支持多项目管理
- 实现材料、分包、合同、结算全流程管理
- 提供数据分析和决策支持

**适用对象：**
- 工程项目管理人员
- 材料采购人员
- 分包商管理人员
- 财务结算人员
- 系统管理员

### 1.2 主要功能模块

| 模块名称 | 功能描述 | 主要用户 |
|----------|----------|----------|
| 项目管理 | 项目信息管理、预算跟踪、成本统计 | 项目经理、管理员 |
| 材料管理 | 材料档案、材料计划、采购申请 | 材料员、采购员 |
| 分包管理 | 分包商档案、分包清单、进度计量 | 分包商、项目经理 |
| 合同管理 | 合同档案、合同清单、状态跟踪 | 合同管理员 |
| 统计分析 | 报表生成、图表分析、操作日志 | 管理层、管理员 |
| 系统管理 | 用户管理、系统设置、数据备份 | 系统管理员 |

### 1.3 系统特点

**1. 功能全面**
- 覆盖工程项目管理全流程
- 集成材料、分包、合同、结算四大核心模块
- 支持多项目并行管理

**2. 操作简便**
- 直观的用户界面设计
- 统一的筛选和搜索功能
- 批量导入导出支持

**3. 数据安全**
- 基于角色的权限控制
- 完整的操作日志记录
- 定期数据备份机制

**4. 扩展性强**
- 模块化设计，便于功能扩展
- 支持自定义配置
- 提供API接口

**5. 移动端适配**
- 响应式设计，支持移动设备访问
- 移动端优化界面
- 触摸操作支持

## 2. 系统安装与部署

### 2.1 环境要求

**硬件要求：**
- CPU：2核以上
- 内存：4GB以上
- 硬盘：50GB以上可用空间
- 网络：稳定的网络连接

**软件要求：**
- 操作系统：Ubuntu 20.04/22.04/24.04、CentOS 7
- Python：3.10及以上版本
- 数据库：PostgreSQL 12及以上
- Web服务器：Nginx 1.18及以上
- 应用服务器：Gunicorn 20.0及以上

**浏览器要求：**
- Chrome 90+（推荐）
- Firefox 88+
- Safari 14+
- Edge 90+

### 2.2 安装步骤

#### 2.2.1 开发环境安装

**步骤1：获取项目代码**
```bash
git clone https://github.com/BH4PYF/MiniErp.git
cd MiniErp
```

**步骤2：配置环境变量**
```bash
cp .env.example .env
# 编辑 .env 文件，配置 SECRET_KEY、数据库连接等
```

**步骤3：启动 Docker 服务**
```bash
docker compose up -d
```

**步骤4：初始化数据库**
```bash
docker exec minierp-web python manage.py migrate
docker exec minierp-web python manage.py createsuperuser
```

访问地址：http://127.0.0.1:8000/（开发）或 https://your-domain.com（生产）

#### 2.2.2 生产环境安装

详见 `deploy/DEPLOYMENT_HTTPS.md`。

快速启动：
```bash
sudo cp deploy/minierp.conf /etc/nginx/conf.d/
sudo systemctl restart nginx
docker compose up -d
```

### 2.3 生产环境部署

详细步骤见 `deploy/DEPLOYMENT_HTTPS.md`。

快速检查清单：
```bash
# 服务状态
docker ps
curl https://your-domain.com/health/    # → "healthy"

# 查看日志
docker logs -f minierp-web
```

## 3. 用户角色与权限

### 3.1 角色说明

系统支持4种用户角色，每种角色具有不同的权限：

| 角色 | 权限描述 | 适用人员 |
|------|----------|----------|
| **管理员** | 系统最高权限，可管理所有功能和用户 | 系统管理员、IT负责人 |
| **管理层** | 查看所有数据，审批关键业务 | 公司管理层、项目经理 |
| **供应商** | 查看采购计划，管理发货单 | 材料供应商 |
| **分包商** | 查看分包合同，填报进度计量 | 工程分包商 |

### 3.2 权限矩阵

| 功能模块 | 管理员 | 管理层 | 供应商 | 分包商 |
|----------|--------|--------|--------|--------|
| **项目管理** | 全部 | 查看 | - | - |
| **材料档案** | 全部 | 查看 | - | - |
| **材料计划** | 全部 | 查看 | - | - |
| **采购申请** | 全部 | 审批 | 查看 | - |
| **入库管理** | 全部 | 查看 | - | - |
| **发货管理** | 全部 | 查看 | 全部 | - |
| **分包商档案** | 全部 | 查看 | - | 查看 |
| **分包清单** | 全部 | 查看 | - | 查看 |
| **进度计量** | 全部 | 查看 | - | 填报/查看 |
| **分包结算** | 全部 | 查看 | - | 查看 |
| **合同管理** | 全部 | 查看 | - | 查看 |
| **统计报表** | 全部 | 全部 | - | 查看 |
| **图表分析** | 全部 | 全部 | - | 查看 |
| **操作日志** | 全部 | 查看 | - | - |
| **用户管理** | 全部 | - | - | - |
| **系统设置** | 全部 | - | - | - |

### 3.3 用户管理

#### 3.3.1 用户创建

**管理员创建用户步骤：**
1. 登录系统，进入"用户管理"页面
2. 点击"新增用户"按钮
3. 填写用户信息：
   - 用户名（必填）
   - 姓名（必填）
   - 密码（必填）
   - 角色（必选）
   - 邮箱（可选）
   - 手机号（可选）
4. 点击"保存"按钮

**用户信息说明：**
- **用户名**：登录系统的唯一标识，建议使用工号或英文名
- **密码**：初始密码，用户首次登录后建议修改
- **角色**：决定用户的权限范围
- **关联信息**：供应商用户需关联供应商，分包商用户需关联分包商

#### 3.3.2 用户管理功能

| 功能 | 描述 | 操作权限 |
|------|------|----------|
| **用户列表** | 查看所有用户信息 | 管理员 |
| **新增用户** | 创建新用户账号 | 管理员 |
| **编辑用户** | 修改用户信息 | 管理员 |
| **禁用用户** | 临时停用用户账号 | 管理员 |
| **删除用户** | 永久删除用户账号 | 管理员 |
| **重置密码** | 重置用户登录密码 | 管理员 |
| **角色分配** | 分配或修改用户角色 | 管理员 |

#### 3.3.3 用户组管理

系统支持用户组管理，便于批量权限分配：

**用户组类型：**
1. **项目组**：按项目划分的用户组
2. **部门组**：按部门划分的用户组
3. **角色组**：按角色划分的用户组

**用户组管理功能：**
- 创建用户组
- 添加/移除组成员
- 批量权限分配
- 用户组权限继承

## 4. 系统登录与主界面

### 4.1 登录系统

#### 4.1.1 登录方式

**1. 标准登录**
- 访问系统地址：https://your-domain.com
- 输入用户名和密码
- 点击"登录"按钮

**2. 记住登录状态**
- 勾选"记住我"选项
- 下次访问自动登录
- 有效期30天

**3. 密码找回**
- 点击"忘记密码"链接
- 输入注册邮箱
- 接收重置密码邮件
- 设置新密码

#### 4.1.2 登录安全

**安全措施：**
- 密码错误次数限制：5次
- 锁定时间：5分钟
- 密码复杂度要求：
  - 长度至少12位
  - 包含大小写字母
  - 包含数字
  - 包含特殊字符
- 会话超时：30分钟无操作自动登出

**登录异常处理：**
- 连续登录失败自动锁定
- 异地登录提醒
- 异常登录行为监控

## 5. 项目管理模块

### 5.1 项目档案管理

#### 5.1.1 项目信息维护

**项目基本信息：**
- 项目编号：自动生成（PRJyyyymmdd000x格式）
- 项目名称：必填，支持中英文
- 项目状态：进行中/已完工/暂停
- 项目地址：详细地址信息
- 项目负责人：关联用户
- 开工日期/完工日期：日期选择
- 项目预算：数字格式，支持小数

**项目扩展信息：**
- 项目描述：详细说明
- 项目类型：建筑工程/市政工程/装饰工程等
- 项目规模：建筑面积/合同金额等
- 项目阶段：前期/施工/竣工等

#### 5.1.2 项目操作功能

**主要功能：**
1. **新增项目**
   - 点击"新增项目"按钮
   - 填写项目信息
   - 保存项目档案

2. **编辑项目**
   - 选择项目，点击"编辑"按钮
   - 修改项目信息
   - 保存修改

3. **查看详情**
   - 点击项目名称或"详情"按钮
   - 查看完整项目信息
   - 查看关联数据（合同、计量、结算等）

4. **删除项目**
   - 选择项目，点击"删除"按钮
   - 确认删除操作
   - 注意：已关联数据的项目不能删除

#### 5.1.3 项目筛选与搜索

**筛选功能：**
- 按项目状态筛选
- 按项目类型筛选
- 按负责人筛选
- 按日期范围筛选

**搜索功能：**
- 按项目编号搜索
- 按项目名称搜索
- 按项目地址搜索
- 支持模糊搜索

### 5.2 项目预算管理

#### 5.2.1 预算编制

**预算项目：**
1. **直接费用**
   - 人工费
   - 材料费
   - 机械费
   - 分包费

2. **间接费用**
   - 管理费
   - 利润
   - 税金
   - 其他费用

**预算编制步骤：**
1. 选择项目
2. 进入预算管理页面
3. 添加预算项目
4. 输入预算金额
5. 保存预算方案

#### 5.2.2 预算执行跟踪

**执行情况：**
- 预算总额
- 已发生费用
- 剩余预算
- 执行率

**预警机制：**
- 预算超支预警
- 预算使用率提醒
- 异常支出报警

### 5.3 项目进度跟踪

#### 5.3.1 进度信息

**进度指标：**
- 计划工期
- 实际工期
- 进度百分比
- 关键节点完成情况

**进度更新：**
- 定期更新进度信息
- 上传进度照片
- 记录进度问题

#### 5.3.2 进度报表

**报表类型：**
1. **进度总览**：项目整体进度
2. **节点进度**：关键节点完成情况
3. **进度对比**：计划与实际对比
4. **进度趋势**：进度变化趋势

## 6. 材料管理模块

### 6.1 材料档案管理

#### 6.1.1 材料信息维护

**材料基本信息：**
- 材料编号：自动生成（MATyyyymmdd000x格式）
- 材料名称：必填，支持中英文
- 规格型号：必填，详细规格信息
- 单位：个/米/吨/立方米等
- 标准单价：数字格式，支持小数

**材料扩展信息：**

| 每天 2:00 | `scripts/backup_db.sh` | 数据库备份 → `backups/db-*.sql.gz` |
| 每天 3:00 | `scripts/backup_files.sh` | 项目文件备份 → `backups/files/files-*.tar.gz` |

默认保留 30 天，旧备份自动清理。

  3. 清除浏览器缓存
  4. 尝试其他浏览器

- **解决方法**：
  1. 检查必填字段是否填写
  2. 检查数据格式是否正确
  3. 检查权限是否足够

- **解决方法**：
  1. 检查筛选条件
  2. 检查数据权限
  3. 刷新页面

- **解决方法**：
